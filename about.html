Perkenalkan Saya

Nama saya Purwanto. Saya mahasiswa universitas Pelita Bangsa semester 4. Saya saat ini bekerja sambil kuliah. Saya mengambil jam kuliah malam, di karenakan siangna saya bekerja. Saya mempunyai sebuah website yaitu techcelup.com. Website techcelup.com merupakan website milik saya yang sudah cukup lama umur nya , namun isi atau konten nya masih belum terlalu banyak di karenakan masih malas menulis . Saya bekerja di salah satu perusahaan hosting terbaik Indonesia yaitu di bagian Development. Techcelup.com ini akan saya isi kembali konten nya, tujuannya utama adalah sebagai catatan saya supaya tidak lupa hal-hal apa saja yang sudah pernah di kerjakan dan sekaligus sharing ke teman-teman semuanya. Semoga artikel-artikel yang saya tulis di sini dapat bermanfaat .
